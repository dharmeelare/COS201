COS202 Study Hub: A Comprehensive Desktop Revision Portal
📌 Project Overview
COS202 Study Hub is a centralized, high-efficiency desktop web application designed to streamline preparation, tracking, and content mastery for the COS202 computer science course. Built with a focus on minimalist UI/UX principles, the hub combines localized academic resource hosting with dynamic time-management and progress-tracking features.

The application eliminates the friction of jumping between scattered PDFs and lecture slides by organizing complex course modules, code snippets, and revision materials into a single, cohesive, distraction-free environment.

✨ Key Features
Centralized Notes Repository: Features structured, cleanly extracted HTML modules for core course topics, allowing for fast reading, quick syntax cross-referencing, and seamless navigation.

Dynamic Interactive UI: A modern layout with responsive navigation panels that prioritize scannability and content clarity.

Progress Tracking & Sprints: Integrated schedule workflows tailored for intensive exam preparation, helping maintain a high-velocity study routine.

Client-Side Architecture: Built using pure HTML5, CSS3 styling, and vanilla JavaScript logic for lightning-fast, zero-dependency performance directly in the browser.

🛠️ Tech Stack & Architecture
Structure: Semantic HTML5 utilizing componentized sections for modular notes management.

Styling: Clean, focused CSS layout architecture designed to reduce eye strain during prolonged study sessions.

Logic: Vanilla JavaScript managing state, view toggles, UI interactions, and dynamic content rendering.
